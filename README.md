# Kurulum
```
git clone https://github.com/Bora185/Z-Bomb.git
cd Z-Bomb
pip3 install -r requirements.txt
python3 zbomb.py
```
# Özellikler
- Sonsuz SMS desteği
- Çoklu numara desteği (max 5)
- 52 Servis (%95 başarı oranı)
- Servis güncellemeleri
- Açık kaynak kodlu
- Ultra gizli
- Hızlı
